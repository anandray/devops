Introducing Wolk Protocols for Trustless HTTP Blockchains

Wolk's mission is to develop decentralized web protocols for pure digital freedom. When application code and user data are held by HTTP blockchain nodes everywhere and nowhere, users and developers can trustlessly stay in control of user data and application code, with no intermediaries in between. Instead of a world controlled by a handful of people and companies run by the few that do not deserve our trust, we imagine a world where pure digital freedom is guaranteed by design through trustless open protocols.
